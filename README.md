## What did I use so far
- [Vite](https://vitejs.dev/)

- [Tailwind CSS](https://tailwindcss.com/)
- [Radix UI](https://www.radix-ui.com/)

## Building
```bash
$ git clone https://github.com/ianfss/ritchie.git
$ cd ritchie
$ npm install
$ npm run dev
```
